Clase 17 - Ejercicio 03

Que hace este ejercicio

Tienes una página con una serie de párrafos. Queremos que cada párrafo tenga debajo un Horizontal Rule (HR), para que esté separado del siguiente. Menos el último párrafo al que no le queremos colocar nada debajo.

  • ✔ Describe de forma clara el comportamiento principal del ejercicio y el elemento del DOM sobre el que actua.
  • ✔ Resume los eventos o metodos jQuery implicados, indicando cuando se disparan y que efecto producen.
  • ✔ Indica el resultado esperado al interactuar con la interfaz para poder validar rapidamente si funciona.

Objetivo: practicar la manipulacion del DOM y el manejo de eventos con jQuery mediante una explicacion estructurada del ejercicio.
