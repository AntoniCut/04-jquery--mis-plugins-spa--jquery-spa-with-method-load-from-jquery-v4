    <!--  -----  Features  -----  -->
    <section class="jquery-page__features">

        <h3 class="jquery-page__features-title">Características Principales</h3>

        <div class="jquery-page__features-grid">

            <article class="jquery-page__feature">
                <span class="jquery-page__feature-icon" aria-hidden="true">🔍</span>
                <h4 class="jquery-page__feature-title">Selección del DOM</h4>
                <p class="jquery-page__feature-text">
                    El selector <code>$()</code> usa la misma sintaxis que los selectores CSS para encontrar
                    elementos: <code>$('#id')</code>, <code>$('.clase')</code>,
                    <code>$('div > p')</code>. Devuelve un objeto jQuery con todos los métodos disponibles.
                </p>
            </article>

            <article class="jquery-page__feature">
                <span class="jquery-page__feature-icon" aria-hidden="true">✏️</span>
                <h4 class="jquery-page__feature-title">Manipulación del DOM</h4>
                <p class="jquery-page__feature-text">
                    Métodos como <code>.html()</code>, <code>.text()</code>, <code>.val()</code>,
                    <code>.append()</code>, <code>.prepend()</code>, <code>.remove()</code> y
                    <code>.addClass()</code> permiten modificar el contenido y los atributos
                    del DOM de forma concisa.
                </p>
            </article>

            <article class="jquery-page__feature">
                <span class="jquery-page__feature-icon" aria-hidden="true">🖱️</span>
                <h4 class="jquery-page__feature-title">Eventos</h4>
                <p class="jquery-page__feature-text">
                    <code>.on()</code>, <code>.off()</code> y <code>.trigger()</code> ofrecen un
                    sistema de eventos unificado. El patrón <em>event delegation</em> con
                    <code>$(parent).on('click', '.hijo', fn)</code> es especialmente potente
                    para elementos dinámicos.
                </p>
            </article>

            <article class="jquery-page__feature">
                <span class="jquery-page__feature-icon" aria-hidden="true">✨</span>
                <h4 class="jquery-page__feature-title">Animaciones</h4>
                <p class="jquery-page__feature-text">
                    Métodos como <code>.fadeIn()</code>, <code>.fadeOut()</code>,
                    <code>.slideUp()</code>, <code>.slideDown()</code> y <code>.animate()</code>
                    facilitan la creación de efectos visuales con control de duración,
                    easing y callbacks al completarse.
                </p>
            </article>

            <article class="jquery-page__feature">
                <span class="jquery-page__feature-icon" aria-hidden="true">🌐</span>
                <h4 class="jquery-page__feature-title">AJAX</h4>
                <p class="jquery-page__feature-text">
                    <code>$.ajax()</code>, <code>$.get()</code> y <code>$.post()</code> abstraen
                    el <code>XMLHttpRequest</code> nativo, devolviendo un objeto <em>deferred</em>
                    compatible con promesas para encadenar <code>.then()</code>, <code>.done()</code>
                    y <code>.fail()</code>.
                </p>
            </article>

            <article class="jquery-page__feature">
                <span class="jquery-page__feature-icon" aria-hidden="true">🔗</span>
                <h4 class="jquery-page__feature-title">Chaining</h4>
                <p class="jquery-page__feature-text">
                    La mayoría de los métodos jQuery devuelven el mismo objeto jQuery,
                    permitiendo encadenar llamadas:
                    <code>$('p').addClass('active').fadeIn(300).css('color', '#21759b')</code>,
                    lo que hace el código más legible y compacto.
                </p>
            </article>

        </div>
