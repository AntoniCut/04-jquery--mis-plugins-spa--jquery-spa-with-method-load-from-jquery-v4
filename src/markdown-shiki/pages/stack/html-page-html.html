  <!--  -----  Intro  -----  -->
  <section class="html-page__intro">
    <p class="html-page__intro-text">
      <strong>HTML</strong> (HyperText Markup Language) es el lenguaje de
      marcado estándar para crear páginas web. Define la estructura y el
      significado del contenido web utilizando etiquetas que describen distintos
      tipos de contenido: párrafos, encabezados, enlaces, imágenes, listas y
      más.
    </p>
    <p class="html-page__intro-text">
      <strong>HTML5</strong>, la versión más reciente del estándar, introduce
      elementos semánticos, APIs nativas del navegador, soporte para audio y
      vídeo sin plugins, y formularios mejorados. Es la base imprescindible de
      cualquier desarrollador web moderno.
    </p>
  </section>

  <!--  -----  Features  -----  -->
  <section class="html-page__features">
    <h3 class="html-page__features-title">Características Principales</h3>

    <div class="html-page__features-grid">
      <article class="html-page__feature">
        <span class="html-page__feature-icon" aria-hidden="true">🏗️</span>
        <h4 class="html-page__feature-title">HTML Semántico</h4>
        <p class="html-page__feature-text">
          Etiquetas como <code>&lt;header&gt;</code>, <code>&lt;main&gt;</code>,
          <code>&lt;section&gt;</code>, <code>&lt;article&gt;</code> y
          <code>&lt;footer&gt;</code>
          dan significado al contenido, mejorando la accesibilidad y el SEO.
        </p>
      </article>

      <article class="html-page__feature">
        <span class="html-page__feature-icon" aria-hidden="true">📝</span>
        <h4 class="html-page__feature-title">Formularios Avanzados</h4>
        <p class="html-page__feature-text">
          HTML5 añade nuevos tipos de input: <code>email</code>,
          <code>date</code>, <code>range</code>, <code>color</code> y
          <code>tel</code>, junto con validaciones nativas del navegador sin
          necesidad de JavaScript.
        </p>
      </article>

      <article class="html-page__feature">
        <span class="html-page__feature-icon" aria-hidden="true">🎬</span>
        <h4 class="html-page__feature-title">Multimedia Nativa</h4>
        <p class="html-page__feature-text">
          Las etiquetas <code>&lt;video&gt;</code>, <code>&lt;audio&gt;</code> y
          <code>&lt;canvas&gt;</code> permiten integrar contenido multimedia sin
          necesidad de plugins externos como Flash.
        </p>
      </article>

      <article class="html-page__feature">
        <span class="html-page__feature-icon" aria-hidden="true">🌐</span>
        <h4 class="html-page__feature-title">APIs del Navegador</h4>
        <p class="html-page__feature-text">
          HTML5 habilita Geolocalización, Web Storage, Drag &amp; Drop, Web
          Workers y Server-Sent Events, potenciando las aplicaciones web
          modernas.
        </p>
      </article>

      <article class="html-page__feature">
        <span class="html-page__feature-icon" aria-hidden="true">♿</span>
        <h4 class="html-page__feature-title">Accesibilidad (a11y)</h4>
        <p class="html-page__feature-text">
          Los atributos <code>aria-*</code>, roles ARIA y el correcto uso de
          etiquetas semánticas garantizan que las páginas sean accesibles para
          todos los usuarios y tecnologías de asistencia.
        </p>
      </article>

      <article class="html-page__feature">
        <span class="html-page__feature-icon" aria-hidden="true">📐</span>
        <h4 class="html-page__feature-title">Estructura del Documento</h4>
        <p class="html-page__feature-text">
          Todo documento HTML se construye con
          <code>&lt;!DOCTYPE html&gt;</code>, <code>&lt;html&gt;</code>,
          <code>&lt;head&gt;</code> y <code>&lt;body&gt;</code>, la base sólida
          de cualquier página web.
        </p>
      </article>
    </div>
  </section>
