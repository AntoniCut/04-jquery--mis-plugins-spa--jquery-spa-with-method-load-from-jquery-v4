Clase 18 - Enviar Datos POST Sin Formularios

Que hace este ejercicio

Este ejercicio demuestra cómo enviar datos al servidor mediante el método POST de AJAX sin necesidad de un formulario HTML. Se utiliza el método $.post() de jQuery para enviar datos directamente desde un enlace.

  • Al hacer clic en el enlace, se envían datos estáticos al servidor mediante AJAX sin recargar la página.
  • Se muestra un indicador de carga mientras se procesa la petición y se recibe la respuesta del servidor.
  • La respuesta del servidor se inserta dinámicamente en el contenedor de salida.

Objetivo: Aprender a enviar datos POST al servidor usando jQuery AJAX sin depender de formularios HTML, controlando completamente el proceso de petición y respuesta.
