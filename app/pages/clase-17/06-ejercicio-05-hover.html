Clase 17 - Ejercicio 05 Hover

Que hace este ejercicio

Tienes una lista de imágenes. Las imágenes tienen atributo "alt", con texto alternativo. Quieres crear pies de foto para cada una de las imágenes. El texto alternativo debe aparecer al hacer hover sobre la imagen, como un pie de foto. El pie de foto debe aparecer debajo de cada imagen. El texto del pie de foto debe de ser el mismo que hay en el alt, el suyo de cada imagen. Haz un tratamiento por si la imagen no tiene atributo alt. En ese caso se debe mostrar un pie de foto con un texto genérico, por ejemplo "copyright 2014".

Esta es una alternativa de solución al ejercicio 05. utilizando los eventos mouseenter y mouseleave .

  • ✔ Describe de forma clara el comportamiento principal del ejercicio y el elemento del DOM sobre el que actua.
  • ✔ Resume los eventos o metodos jQuery implicados, indicando cuando se disparan y que efecto producen.
  • ✔ Indica el resultado esperado al interactuar con la interfaz para poder validar rapidamente si funciona.

Objetivo: practicar la manipulacion del DOM y el manejo de eventos con jQuery mediante una explicacion estructurada del ejercicio.
