            <article class="css-page__feature">

                <h4 class="css-page__feature-title">Selectores y Especificidad</h4>
                <p class="css-page__feature-text">
                    CSS ofrece selectores de elemento, clase (<code>.clase</code>), ID (<code>#id</code>),
                    atributo, pseudoclases (<code>:hover</code>, <code>:nth-child</code>) y
                    pseudoelementos (<code>::before</code>, <code>::after</code>).
                </p>
            </article>

            <article class="css-page__feature">
                <span class="css-page__feature-icon" aria-hidden="true">📦</span>
                <h4 class="css-page__feature-title">Box Model</h4>
                <p class="css-page__feature-text">
                    Todo elemento en CSS es una caja compuesta por <code>content</code>,
                    <code>padding</code>, <code>border</code> y <code>margin</code>.
                    La propiedad <code>box-sizing: border-box</code> simplifica el cálculo de tamaños.
                </p>
            </article>

            <article class="css-page__feature">
                <span class="css-page__feature-icon" aria-hidden="true">🔀</span>
                <h4 class="css-page__feature-title">Flexbox</h4>
                <p class="css-page__feature-text">
                    Flexbox permite alinear y distribuir elementos en una dimensión (fila o columna)
                    con propiedades como <code>justify-content</code>, <code>align-items</code>
                    y <code>flex-wrap</code>.
                </p>
            </article>

            <article class="css-page__feature">
                <span class="css-page__feature-icon" aria-hidden="true">⚏</span>
                <h4 class="css-page__feature-title">CSS Grid</h4>
                <p class="css-page__feature-text">
                    Grid Layout ofrece un sistema de diseño bidimensional poderoso,
                    con <code>grid-template-columns</code>, <code>grid-template-rows</code>
                    y la unidad <code>fr</code> para layouts complejos de forma sencilla.
                </p>
            </article>

            <article class="css-page__feature">
                <span class="css-page__feature-icon" aria-hidden="true">🎨</span>
                <h4 class="css-page__feature-title">Variables CSS (Custom Properties)</h4>
                <p class="css-page__feature-text">
                    Las variables CSS (<code>--nombre: valor</code>) permiten definir valores
                    reutilizables en <code>:root</code> o ámbitos locales,
                    facilitando el theming y el mantenimiento del código.
                </p>
            </article>

            <article class="css-page__feature">
                <span class="css-page__feature-icon" aria-hidden="true">✨</span>
                <h4 class="css-page__feature-title">Animaciones y Transiciones</h4>
                <p class="css-page__feature-text">
                    Con <code>transition</code> y <code>@keyframes</code> + <code>animation</code>,
                    CSS permite crear efectos visuales fluidos y dinámicos sin necesidad
                    de JavaScript.
                </p>
            </article>
